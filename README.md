# Hospital 30-Day Readmission Predictor

Predicts whether a diabetic patient will be readmitted to hospital
within 30 days of discharge. Which is a real problem costing the US $26B/year.

## Dataset
UCI Diabetes 130-US Hospitals (1999-2008)
Download here: https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008
## Results
- ROC-AUC: 0.6428
- Key finding: prior inpatient visits and number of diagnoses
  are the strongest predictors of early readmission

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
